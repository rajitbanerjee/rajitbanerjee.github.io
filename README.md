# [rajitbanerjee.com](https://rajitbanerjee.com)

My portfolio website, built with TypeScript/React.js (Create React App).

## Getting Started

- For development:
  ```bash
  npm i
  npm start
  ```
- For production:

  ```bash
  npm run build
  npm run analyze

  npm install -g serve # optional
  serve -s build
  ```
